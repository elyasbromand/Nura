Project: Nura

Description:
Nura is a web application focused on health assessment and cluster detection, with modules for maternal, neonatal, outbreak, and history tracking. It features assessment flows, cluster mapping, risk analysis, and a modular UI built with Next.js and Tailwind CSS.
